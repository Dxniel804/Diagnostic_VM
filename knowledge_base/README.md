# Knowledge Base - Vendamais

## Como usar a funcionalidade de Knowledge Base

Esta pasta permite armazenar documentos PDF da empresa Vendamais que serão usados para enriquecer as análises estratégicas geradas pela IA.

## O que fazer:

1. **Adicione PDFs**: Coloque na pasta `knowledge_base/` os documentos PDF que contenham:
   - Informações sobre produtos e serviços da Vendamais
   - Propostas comerciais
   - Cases de sucesso
   - Metodologias de vendas
   - Argumentários de vendas
   - Manuais de treinamento
   - Qualquer outro material relevante para as estratégias comerciais

2. **Formato dos arquivos**: Apenas arquivos `.pdf` são suportados

3. **Como funciona**: 
   - Ao iniciar o sistema, todos os PDFs na pasta serão automaticamente lidos
   - O conteúdo é extraído e incorporado ao conhecimento da IA
   - A IA usará esse conhecimento para gerar estratégias mais alinhadas com a Vendamais

4. **Atualização**: 
   - Para adicionar novos documentos, simplesmente coloque os PDFs na pasta
   - Reinicie o sistema para que os novos documentos sejam carregados

## Exemplos de documentos úteis:

- Catálogo de produtos/serviços
- Propostas comerciais padrão
- Estudos de caso de clientes
- Scripts de vendas
- Políticas comerciais
- Diferenciais competitivos

## Importante:

- O sistema limita o conteúdo extraído para não sobrecarregar o prompt da IA
- Documentos muito grandes podem ter seu conteúdo truncado
- Mantenha os PDFs atualizados para garantir análises precisas
